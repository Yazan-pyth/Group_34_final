|--------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------|-------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------|-----|
| MNIST                          | База из 70 000 черно-белых изображений рукописных цифр размером 28x28 пикселей. | Fashion-MNIST                  | Набор из 70 000 изображений предметов одежды и обуви от Zalando, также размером 28x28 пикселей. Создан как более сложная замена MNIST.                                                             | М4                | github.com/zalandoresearch/fashion-mnist                    | MIT License                                                                                                                                                |     |
| COCO                           | Масштабный датасет для задач обнаружения объектов, сегментации и описания изображений. | Google Speech Commands Dataset | Набор из 105 829 однoсeкундных аудиофайлов (v0.02) на английском языке, содержащих одно произнесенное слово. Включает 35 командных слов.                                                           | М5                | tensorflow.org/datasets/catalog/speech_commands             | Creative Commons BY 4.0                                                                                                                                    |     |
| UCI Gesture Recognition        | Датасет для сегментации фаз жестов.
