# Контроль качества репозитория

## Проверка структуры

- [ ] В корне присутствуют только каталоги и файлы, предусмотренные шаблоном.
- [ ] Для каждого из шести модулей создан отдельный каталог.
- [ ] Ссылки в `README.md` открывают существующие файлы.
- [ ] В документах отсутствуют незаполненные шаблонные маркеры.

## Проверка РПД

- [ ] Общая трудоёмкость составляет 108 академических часов.
- [ ] Сумма часов шести модулей составляет 98 часов, итоговая аттестация — 10 часов.
- [ ] Лабораторный практикум содержит 12 работ общей трудоёмкостью 28 часов.
- [ ] Указаны индикаторы `PL-4.2`, `DL-1.11`, `LC-5.1`, `DL-3.3`.
- [ ] Формы контроля соответствуют РПД: текущая, рубежная и промежуточная аттестация.

## Проверка КИМ

- [ ] В каждом модуле приведены КИМ соответствующих лабораторных работ.
- [ ] В каждом модуле присутствует `tests.md` с тестами только по лекциям этого модуля, а `README.md` содержит рабочую ссылку на него.
- [ ] Продолжительность работ соответствует РПД.
- [ ] Требуемые метрики, оборудование, порядок выполнения и результаты отчёта сохранены.
- [ ] Критерии итогового проекта соответствуют РПД.


## Проверка добавленных методических материалов

- [ ] В каждом модуле присутствует `independent-work.md`.
- [ ] В `Exam/` присутствуют методические указания по подготовке к итоговой защите.
- [ ] Система оценивания соответствует весам 30% / 20% / 50% и порогу 60 из 100.
- [ ] В `resources/` присутствуют учебники, научные статьи, датасеты и программные средства.
- [ ] В `team/README.md` указаны организация, роль, зона ответственности, контакты и профиль каждого участника.
- [ ] В `other/llm-prompt.md` размещён вспомогательный промпт для работы с базой знаний без подмены нормативных источников.
