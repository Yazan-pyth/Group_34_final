# Вклад в репозиторий

1. Создайте отдельную ветку для изменения.
2. Изменяйте только материалы, относящиеся к РПД, КИМ, ресурсам или методическим рекомендациям.
3. Не меняйте корневую структуру репозитория без согласования с командой.
4. Проверьте ссылки, часы, коды компетенций и отсутствие незаполненных шаблонных полей.
5. Создайте pull request и кратко укажите, какие разделы РПД или КИМ были изменены.
6. После проверки изменения объединяются с основной веткой.
